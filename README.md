# Space defender

Tématem je klasický Space defender typ hry, hráč se bude pohybovat po mapě a střílet objekty-nepřátele a překážky

- Cíle projektu:
    - Zpracovat pohyb hráče
    - Načtení mapy
    - Zásahy, poškození, srážka objektu
    - Zaznamenávání nejvyššího skóre
    - Herní UI
